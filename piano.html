Instrucciones para tocar el piano

Las teclas pueden ser tocadas haciendo click. Tambien puedes tocar con el teclado usando las teclas de las letras para las teclas azul cielo y los numeros para las teclas azul oscuro. Como puedes observar, en cada tecla esta escrita su nota correspondiente.
